📘 Spiel Erklärung

🌱 Pflanzen Button: Startet eine neue Pflanze.

🔄 Revive: Reset der Sammlung + neue Pflanzen freischalten.

🌤️ Wetter: Ändert Wachstums-Speed (Sonne schneller, Regen langsamer).

🌙 Tageszeit: Tag = schneller Wachstum, Nacht = langsamer.

❗ Seltenheit: Common → Mythic beeinflusst Zeit & Wert.

📦 Einklappen: Jede Seltenheit kann auf/zu geklickt werden.
